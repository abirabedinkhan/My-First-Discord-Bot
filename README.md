# My First Discord Bot

It is my Discord Bot (Made with Python, Discord.py)


## Installation

Install my-project with pip3

```bash
  cd my-project
  pip3 install -r requirements.txt
```
Then replace 'TOKEN' from bot.py with your TOKEN

```bash
  python3 .\bot.py
```
